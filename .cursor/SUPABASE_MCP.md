# Supabase MCP

This project includes Supabase MCP at `.cursor/mcp.json` (project: `ontajayxgjpvhxahvncj`).

After opening this workspace:

1. **Reload Cursor** (Cmd+Shift+P → "Developer: Reload Window") so MCP servers load.
2. Open **Cursor Settings → MCP** and confirm **supabase** is connected (authenticate if prompted).
3. Ask the agent to run `apply_migration` with `supabase/migrations/20260703220000_white_eagle_cleanup_and_schema.sql`.

Alternative without MCP: paste `supabase/schema.sql` into the [Supabase SQL Editor](https://supabase.com/dashboard/project/ontajayxgjpvhxahvncj/sql/new).
